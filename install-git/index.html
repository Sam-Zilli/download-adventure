Install Git with Homebrew

Git is the software that lets you track changes and sync code with GitHub.

Let's use Homebrew to install Git. Do you remember the command to install something with Homebrew?

Now run this in your Terminal:

brew install git

This may take a minute to download and install.

Verify Git is Installed

Check that Git installed correctly by typing:

git --version

You should see something like git version 2.43.0 (your numbers may differ).

Configure Git

Git needs to know who you are so it can label your commits. Run these two commands in Terminal (replace with your actual name and email):

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Use the email associated with your GitHub account.

You only need to do this once on your computer.
