Terminal Command Cheat Sheet

This cheat sheet contains essential terminal commands to help you navigate and manage files from the terminal.

Essential Terminal Commands
  • pwd Prints the current working directory (where you are in the file system).
  • ls Lists all files and directories in the current directory.
  • cd ~ Changes to the home directory of the user.
  • cd .. Moves up one directory level.
  • cd <directory> Changes the current directory to the specified <directory>.
  • clear Clears the terminal screen.
  • echo “what to echo” Prints the text inside the quotes to the terminal.
  • mkdir <folder_name> Creates a new directory with the specified <folder_name>.
  • rmdir <folder_name> Removes an empty directory with the specified <folder_name>.
  • less <file_name> Displays the contents of a file one page at a time. Press q to quit.
  • touch <file.txt> Creates an empty file with the specified name.
  • rm <file.txt> Deletes the specified file.
  • grep “search terms” <file_to_search> Searches for the specified search terms in the given file and displays matching lines.
  • mv <thing> <newLocation> Moves a file or directory to a new location.
  • mv <thing> <newNameForThing> Renames a file or directory to the specified new name.
