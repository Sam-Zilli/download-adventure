Why Disable AI Help?

Important for Learning

AI coding tools like GitHub Copilot can write code for you. While these are powerful tools, they prevent you from learning.

You need to learn to code yourself first before using AI tools.

What We're Disabling:

GitHub Copilot - AI that writes code for you

IntelliCode AI - AI-powered autocomplete suggestions

Inline Suggestions - Gray text that appears as you type

Don't worry - you'll still have basic autocomplete and syntax highlighting. You're just turning off the AI that does the thinking for you.

Turn Off AI Features

Follow these steps in VS Code:

  1. Open VS Code
  2. Press Cmd + , to open Settings
  3. In the search bar at the top, type: copilot
  4. If you see any Copilot settings, uncheck all of them or disable Copilot
  5. Now search for: inlineSuggest
  6. Find "Editor › Inline Suggest: Enabled" and uncheck it
  7. Search for: intellicode
  8. If you see "Python › Analysis: Type Checking Mode", you can leave it as is
  9. Look for any settings with "AI" or "predict" and disable them

✓ What You Should See:

When typing code, you should NOT see gray "ghost text" appearing automatically.

You SHOULD still see a dropdown menu with suggestions when you type - that's normal autocomplete (not AI).
