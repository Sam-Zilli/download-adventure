Commands with FLAGS

Flags are special arguments that start with - or -- and modify how a command behaves:

ls -la

■ Command = ls (list files)
■ Flag = -la (list all files in long format)

Single Dash -: Short flags (one letter). Can be combined like -la = -l -a
Double Dash --: Long flags (full words) like --help or --version

Common Flags You'll See:

-l = Long format (shows detailed file information)

-a = All files (including hidden files that start with .)

-r = Recursive (work on folders and everything inside them)
