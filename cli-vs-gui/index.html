Learn to identify Command Line vs Graphical Interfaces

Two Ways to Use Computers

There are two main ways to interact with software:

CLI (Command Line Interface)

You type text commands to tell the computer what to do.

Example: ls to list files

Fast and powerful
Great for automation
Uses keyboard only

GUI (Graphical User Interface)

You click buttons, menus, and icons to tell the computer what to do.

Example: Clicking a folder to open it

Easy to learn
Visual feedback
Uses mouse and keyboard
