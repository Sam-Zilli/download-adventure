What is VS Code?

VS Code is a code editor - a place to write, edit, and organize your code.

We could write code in Terminal, but VS Code gives us helpful tools like:

  • Syntax Highlighting - colors make code easier to read
  • Autocomplete - suggests code as you type
  • Error Detection - shows mistakes before you run
  • File Explorer - organize your project files

Install VS Code

Remember Homebrew? Let's use it to install VS Code.

  1. Go to the Homebrew page for VS Code
  2. Find the install command and copy it
  3. Paste it into your Terminal and press Enter
  4. Wait for it to install (this may take a minute)
Hint: The command will look something like brew install --cask visual-studio-code

Verify Installation

Let's make sure VS Code installed correctly.

Type this in your Terminal:

code -v

You should see a version number like 1.85.0 (your numbers may differ).

VS Code is now an app installed on your computer. You can put it in your dock for easy access.
