What is Git Clone?

Clone means to make a copy. When you clone a repository, you're downloading a copy of the code from GitHub to your computer.

The Command

In the terminal, cloning looks like this:

git clone https://github.com/username/repository-name.git

This downloads the entire repository to your computer, including all the files and the complete history of changes.

Why Clone?

Cloning lets you:

  • Work on code offline, on your own computer
  • Make changes and test them locally
  • Push your changes back to GitHub when ready
  • Collaborate with others on the same project
