Git and GitHub Walkthrough

Step 1: Create a New Repository on GitHub

Let's create a brand new repository from scratch!

  1. Go to GitHub.com and make sure you're logged in
  2. Click the green "New" button or the "+" icon in the top right corner
  3. Select "New repository"
  4. Give your repository the name Intro-To-Programming
  5. Add a description (optional but recommended)
  6. Choose visibility PRIVATE
  7. Check "Add a README file" - this creates an initial file in your repo
  8. Click "Create repository"

Step 2: Add Your Instructor as a Collaborator

Now you need to add your instructor as a collaborator so they can access your private repository:

  1. On your repository page, click on the "Settings" tab
  2. In the left sidebar, click on "Collaborators"
  3. Click the green "Add people" button
  4. In the search box, type: Sam-Zilli
  5. Click on the Sam-Zilli user when it appears
  6. Click "Add Sam-Zilli to this repository"
  7. Leave the role as "Write" (this is the default)

Now get the clone URL:

  1. Go back to your repository's main page (click on "Code" tab)
  2. Click the green "Code" button
  3. Make sure "HTTPS" is selected
  4. Click the copy button to copy the repository URL to your clipboard

Step 3: Clone Your Repository to Your Computer

Now let's use the terminal to clone (download) your repository to your computer:

  1. Open your terminal application
  2. Navigate to where you want to store your project. For example:
    cd ~/Documents
    or
    cd ~/Desktop
  3. Clone your repository using the URL you copied:
git clone <paste-your-repository-url-here>

Example:

git clone https://github.com/yourusername/Intro-To-Programming.git

Press Enter and watch Git download your repository!

Step 4: Explore Your Cloned Repository

Let's verify the clone worked and explore your repository using terminal commands:

  1. List the contents of your current directory:
    ls
    You should see your repository folder!
  2. Navigate into your repository:
    cd Intro-To-Programming
    (This should match your repository name)
  3. List the contents of your repository:
    ls
    You should see your README.md file!
  4. See ALL files, including hidden ones:
    ls -a
    Notice the .git folder - this is what makes it a Git repository!
  5. Check your repository's status:
    git status
    This shows you're on the main branch and everything is up to date.
  6. View your README file:
    cat README.md
    This displays the contents of your README file in the terminal.

Congratulations! You've successfully created a GitHub repository and cloned it to your computer using terminal commands!
