Final Boss: Submit Your Scratch Project

Download, push to GitHub, and submit on Canvas

Step 1 of 6

Step 1: Download Your Scratch Project

First, let's download your Scratch project as a file.

  1. Go to scratch.mit.edu and sign in
  2. Open your project (the one you want to submit)
  3. Click File in the menu bar
  4. Click Save to your computer
  5. Save the .sb3 file somewhere you can find it (like Downloads or Desktop)
What's an .sb3 file? This is a Scratch project file. It contains all your code, sprites, and sounds in one file.

Step 2: Add the File to Your Repository

Now move the .sb3 file into your GitHub repository folder.

  1. Open Finder
  2. Find your downloaded .sb3 file
  3. Drag it into your Intro-To-Programming folder
Where's my repo folder? It's wherever you cloned it earlier - probably in Documents or Desktop. You can also see it in VS Code's Explorer panel.
Make sure the file is directly inside your repo folder, not in a subfolder (unless your instructor said otherwise).

Step 3: Push to GitHub

Now let's upload your Scratch file to GitHub using the workflow you learned.

  1. Open your repo in VS Code
  2. Click the Source Control icon (left sidebar)
  3. You should see your .sb3 file under "Changes"
  4. Click the + button to stage it
  5. Type a commit message like: Add Scratch project
  6. Click Commit
  7. Click Sync Changes to push to GitHub
Remember: Add → Commit → Push

Step 4: Verify on GitHub

Let's make sure your file made it to GitHub.

  1. Go to github.com
  2. Navigate to your Intro-To-Programming repository
  3. Look for your .sb3 file in the file list
  4. Click on it to make sure it's there

Before continuing, verify:

  • Your .sb3 file appears in your repository on GitHub
  • The file name looks correct
Don't see your file? Go back and make sure you clicked "Sync Changes" after committing. The file needs to be pushed to GitHub, not just committed locally.

Step 5: Copy Your Repository Link

Now get the link to submit on Canvas.

  1. On your GitHub repository page, look at the URL in your browser
  2. It should look like: https://github.com/yourusername/Intro-To-Programming
  3. Copy this URL from your browser's address bar
This is the link you'll submit on Canvas. It's the main page of your repository where your instructor can see all your files.

Step 6: Submit on Canvas

Finally, submit your repository link on Canvas.

  1. Go to Canvas and find the assignment
  2. Click Submit Assignment
  3. Choose Website URL as the submission type
  4. Paste your GitHub repository link
  5. Click Submit

Final checklist:

  • Your .sb3 file is visible on GitHub
  • You submitted the correct repository link on Canvas
  • Canvas shows your submission was received

Submission Complete

You've successfully submitted your Scratch project via GitHub.

This is the same workflow you'll use for future assignments:

Download → Add to Repo → Push → Submit Link
