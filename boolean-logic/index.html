How Logic Gates Work:

  • AND Gate: Output is ON only when BOTH inputs are ON
  • OR Gate: Output is ON when AT LEAST ONE input is ON
  • XOR Gate: Output is ON when EXACTLY ONE input is ON (exclusive or)
