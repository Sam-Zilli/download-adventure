Commands with MULTIPLE Arguments

Some commands need MORE THAN ONE piece of information and therefore get multiple arguments. Arguments are separated by spaces.

cp file.txt backup.txt

■ Command = cp (copy)
■ Argument 1 = file.txt (what to copy)
■ Argument 2 = backup.txt (where to copy it)

mkdir folder1 folder2

■ Command = mkdir (make directory)
■ Argument 1 = folder1 (first folder to create)
■ Argument 2 = folder2 (second folder to create)
