Install Python Extension

Step 1 of 3

Open VS Code

First, let's open VS Code.

  1. Press Cmd + Space to open Spotlight
  2. Type Visual Studio Code
  3. Press Enter to open it

You should see the VS Code welcome screen.

Open the Extensions Panel

VS Code has extensions that add extra features. We need one for Python.

  1. In VS Code, go to View → Extensions
  2. Or click the Extensions icon in the left sidebar (it looks like 4 squares)

Install the Python Extension

  1. In the search bar, type Python
  2. Find the one by Microsoft (it has millions of downloads)
  3. Click the blue Install button
What does this extension do? It helps VS Code understand Python code - giving you syntax highlighting, error detection, and autocomplete.
