Install GitHub Desktop with Homebrew

GitHub Desktop is a visual app that makes it easy to save your code to GitHub.

Let's use Homebrew to install GitHub Desktop. Do you remember the command to install something with Homebrew?

Now run this in your Terminal:

brew install --cask github

This may take a minute to download and install.

Open GitHub Desktop

GitHub Desktop should now be installed. Let's open it:

  1. Press Cmd + Space to open Spotlight
  2. Type GitHub Desktop
  3. Press Enter to open it

You should see the GitHub Desktop welcome screen.

Sign In to GitHub

GitHub Desktop needs to connect to your GitHub account.

  1. In GitHub Desktop, click Sign in to GitHub.com
  2. Your web browser will open - sign in with your GitHub username and password
  3. Click Authorize desktop to give GitHub Desktop permission
  4. Return to GitHub Desktop

GitHub Desktop will automatically configure everything for you - no terminal commands needed!
