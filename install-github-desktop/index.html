What is GitHub Desktop?

GitHub Desktop is a visual app that makes it easy to save your code to GitHub.

Instead of using terminal commands, you'll use buttons and menus to save and upload your work.

Download GitHub Desktop

  1. Go to desktop.github.com
  2. Click Download for macOS
  3. The .zip file will download to your Downloads folder

Install GitHub Desktop

  1. Go to your Downloads folder and find GitHubDesktop.zip
  2. Double-click the zip file to unzip it
  3. You'll see an app called GitHub Desktop
  4. Drag that app to your Applications folder
  5. Open it from Applications (or Spotlight: Cmd + Space, type "GitHub Desktop")

Sign In to GitHub

GitHub Desktop needs to connect to your GitHub account.

  1. In GitHub Desktop, click Sign in to GitHub.com
  2. Your web browser will open - sign in with your GitHub username and password
  3. Click Authorize desktop to give GitHub Desktop permission
  4. Return to GitHub Desktop

GitHub Desktop will automatically configure everything for you - no terminal commands needed!

GitHub Desktop Installed

You now have GitHub Desktop on your computer.
