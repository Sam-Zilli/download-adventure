What is VS Code?

VS Code (or Visual Studio Code) is an IDE - a place to write, edit, and organize your code.

We could write code in Terminal, but VS Code gives us helpful tools like:

  • Syntax Highlighting - colors make code easier to read
  • Autocomplete - suggests code as you type
  • Error Detection - shows mistakes before you run
  • File Explorer - organize your project files

Download VS Code

  1. Go to code.visualstudio.com
  2. Click the big Download for Mac button
  3. The .zip file will download to your Downloads folder
Note: The website automatically detects your Mac type (Intel or Apple Silicon) and gives you the right version. If it asks, you can check yours by running uname -m in Terminal (arm64 = Apple Silicon, x86_64 = Intel).

Install VS Code

  1. Go to your Downloads folder and find VSCode-darwin.zip
  2. Double-click the zip file to unzip it
  3. You'll see an app called Visual Studio Code
  4. Drag that app to your Applications folder
  5. Open it from Applications (or Spotlight: Cmd + Space, type "Visual Studio Code")
Tip: You can drag VS Code to your Dock for easy access later.

VS Code Ready

You now have a professional code editor installed.
