Clone a GitHub Repo with VS Code

Using the built-in Source Control features

Step 1 of 5

Step 1: Create a New Repository on GitHub

First, let's create a repository on GitHub.

  1. Go to GitHub.com and make sure you're logged in
  2. Click the green "New" button or the "+" icon in the top right
  3. Select "New repository"
  4. Name your repository: Intro-To-Programming
  5. Set visibility to PRIVATE
  6. Check "Add a README file"
  7. Click "Create repository"

Step 2: Add Your Instructor as a Collaborator

Add your instructor so they can access your private repository:

  1. On your repository page, click "Settings"
  2. In the left sidebar, click "Collaborators"
  3. Click the green "Add people" button
  4. Search for: Sam-Zilli
  5. Click on the user and then "Add Sam-Zilli to this repository"

Step 3: Copy the Clone URL

Now get the URL for your repository:

  1. Go to your repository's main page (click the "Code" tab)
  2. Click the green "Code" button
  3. Make sure "HTTPS" is selected
  4. Click the copy button to copy the URL
Tip: The URL will look something like:
https://github.com/yourusername/Intro-To-Programming.git

Step 4: Clone in VS Code

Now let's clone the repository using VS Code:

  1. Open VS Code
  2. Press Cmd + Shift + P to open the Command Palette
  3. Type "Git: Clone" and select it
  4. Paste your repository URL and press Enter
  5. Choose where to save the folder (like your Documents folder)
  6. When prompted, click "Open" to open the cloned repository
First time? VS Code may ask you to sign in to GitHub. Click "Allow" and follow the prompts in your browser.

Step 5: Explore Your Repository

You should now see your repository open in VS Code.

  1. In the Explorer panel (left side), you'll see your files
  2. Click on README.md to view it
  3. Notice the Source Control icon on the left sidebar (it looks like a branch)
What just happened? VS Code downloaded your repository from GitHub to your computer. Any changes you make can be "pushed" back to GitHub using the Source Control panel.

You Did It!

You've successfully cloned a GitHub repository using VS Code.
