Install Python

Step 1 of 3

What is Python?

Python is a programming language. You'll use it to write code that tells the computer what to do.

We need to download Python so your computer can understand and run Python programs.

Download Python

  1. Go to python.org/downloads/
  2. Click the big Download Python button (it will show the latest version)
  3. The installer file will download to your Downloads folder
Note: The website automatically detects your Mac type. If it asks, you can check yours by running uname -m in Terminal (arm64 = Apple Silicon, x86_64 = Intel).

Install Python

  1. Open the .pkg file from your Downloads folder
  2. Follow the installer steps (just keep clicking "Continue" and "Agree")
  3. IMPORTANT: Make sure to check the box that says "Add Python to PATH" if you see it
  4. Enter your Mac password when prompted
  5. Click "Install" and wait for it to finish

Verify It Worked

Open Terminal and type:

python3 --version

You should see something like Python 3.12.0 (your numbers may differ).

If it says "command not found": Close Terminal completely (Cmd + Q), then open it again and try the command.

Python Installed

You can now write and run Python programs on your computer.
