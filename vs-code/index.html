Visual Studio Code (VS Code) - A Beginner's Guide

Technically we could write all of our code in Terminal - but wouldn't it be nice to have a pretty interface with helpful tools?

Writing code in Terminal is like writing a book in the old Notepad application that came with computers. VS Code is like Word - many more features.

VS Code is an IDE - Integrated Development Enviornment. All the things developers need to do - write code, debug, project management is all included.

How to Install VS Code

Anyone remember that tool we used to install things? Oh yeah, brew...

  1. Go to the brew page for VSCode.
  2. Copy the command into your Terminal to install VS Code.
  3. Finally, check that it worked by entering: % code -v
